# vimrc
My vimdc

## Install

1. Install vimrc
    
    git clone git@github.com:ovsoil/vimrc.git
    cp vimrc/vimrc ~/.vimrc

2. Install Vundle
        
    git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

3. Install Plugin
    
    Launch `vim` and run `:PluginInstall`
